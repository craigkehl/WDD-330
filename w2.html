Topics studied

  1. Functions: Types, structure and declaration
    2. Standard Function
    function var_name(pram1, pram2, ...pram) { 
       z = a + b;
       return z;
    }
    Advantages:
    • It can be very concise
    Function Expression
    const name = function(pram1, pram2, ...pram) { return z}
    Advantages:
    • It can be very concise
    Disadvantages:
    • Must be declared before it is used
    Fat Arrow Function - Standard
    const add = (a, b) => { 
      const result = a + b;
      return result;
    };
    Advantages:
    • It can be very concise
    • Semi-colon at end
    • No function keyword
    Fat Arrow Function - shorter with one param
    const log = message => { 
      console.log(message);
    };
    Advantages:
    • You can omit parentheses if there is only one parameter
    Fat Arrow Function - one expression
    const add = (a, b) => a + b;
    Advantages:
    • Very short
    • Expression result is always automatically returned
