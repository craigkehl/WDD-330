Animations and Transitions

Tansition

To begin, you define start and end states attributes and amounts. A transition statement animates the change of states by specifying elements like:

  • Which attribute's state changes to apply
  • Duration of the transition
  • Delay from the triggered change to begin
  • The rate and/steps

Multiple attributes can be assigned. Mulitple transition statements may have different specifications to allow the elements to respond differently. The browser fills in the states in between.

Animation

With an animation, you can specifically define in-between states to control how the animation progresses. Animations are assigned a name and called when a class is applied or enters into a state such as :hover to trigger the animation.

Web Examples/Projects

The following are either projects or examples from web searches I studied to see how to apply this week's material. SVGs are scalable vector graphics:

  • They maintain their quality as their size requirements change.
  • They can also be very light on code weight compare to a bitmap type graphic. However, depending on the graphics complexity, they can also be larger and consume more processing resources.
  • Unlike a bitmap graphic, the elements in the SVG can be individually addressed and modified through XHML and CSS.
  • Those elements can also have event listeners attached to them and act as triggers to execute code.
