Here are different ways to create arrays

Most standard way of creating an array and populating it in the same statement. By passing in more than one number separated by commas populates the array with those numbers

const arr = [1, 2, 3, 4];

The following three ways create empty arrays of length 1. Passing in a single number defines the length but does not populate the array.

const arr1 = [1];

Assumed calling the contructor

const arr2 = new Array(1);

Calls the constructor method with the parameter designating the length.

const arr3 = Array(1);

"new" keyword ommitted

Javascript assumes you are populating and constructing an array if you provide comma separated values.

const arr2a = Array(2, 4, 6);

Because a string is being passed, an array populated with that string is created. Very forgiving.

const arr4 = ['Hi'];

or

const arr5 = Array('Hi');

or

const arr6 = new Array('Hi');

Array.of() calls the Array constructor of (items,). So, both a single number or any item creates an array and populates it with the items listed.

const arr7 = Array.of(1); const arr10 = Array.from(1);

What can arrays hold?

  • A collection of any data type
  • A collection of objects
  • A collection of other arrays
  • Any combination
const hobbies = ['reading', 'fly fishing', 'calculus']; const mixData = [23, 'BMW', {key: 'value'}];

Adding items to an array

push() adds items to the end of an array. While unshift() adds items to the beginning. Open the dev console. Then push the 'Test me' button. Notice the GreatBooks array with three books. push() with then add one book to the end. After that unshift() will add on book to the start.

const greatBooks = ['To Kill A Mocking Bird', 'As a Man Thinketh', 'Make It Stick']; console.log(greatBooks); greatBooks.push('The Goal'); greatBooks.unshift('Good to Great');

Remember: Open the Dev Console to see the arrays.
